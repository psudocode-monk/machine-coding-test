# 🚀 React Machine Coding Hub

A simple React-based platform to practice **real-world machine coding questions**.  
Built to improve problem-solving, UI building, and frontend system thinking.

---

## 📌 Features

- 🔍 Search questions by keyword  
- 🧩 Collection of React machine coding problems  
- ⚡ Clean and minimal UI (Tailwind CSS)  
- 🧭 Client-side routing using React Router  
- 🧱 Component-based scalable structure  

---

## 🛠️ Tech Stack

- React (Vite)
- React Router DOM
- Tailwind CSS
- JavaScript (ES6+)
